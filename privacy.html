Privacy Policy

Last updated: July 2026

1. Information We Collect

When you create an Artispreneur account, we collect your name, email address, and any profile information you provide. If you connect Google Drive, we access only the Artispreneur folder you authorize. If you use the BYOK feature, your API keys are encrypted at rest and never shared.

2. Agent Data

Your agent's state — including soul.md, session memory, decisions, and learnings — is stored in an isolated per-user workspace. No agent data is shared across users. Agent interactions are logged for debugging and improvement purposes.

3. LLM Providers

When using the Free tier, your agent prompts are sent to Google's Gemini API via our provisioned credits. When using BYOK, prompts route directly to your chosen provider using your key — we do not intermediate or log those requests beyond basic usage tracking.

4. Signal / Wire Messaging

If you connect Signal or Wire messaging, messages are end-to-end encrypted via those platforms' native protocols. The bridge service decrypts messages only to route them to your agent instance.

5. Data Retention

Your workspace data is retained as long as your account is active. You can request deletion at any time by contacting us. Deletion removes your agent state, outputs, and workspace files.

6. Third-Party Services

Artispreneur integrates with third-party services including but not limited to: Google (for Gemini API and Drive), Vercel (for hosting), Supabase (for authentication and database), and Signal/Wire (for messaging). Each service has its own privacy policy.

7. Contact

Questions about this policy? Get in touch.
