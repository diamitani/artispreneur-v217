Frequently Asked Questions

What are Gemini API free credits?

Google offers free API credits for Gemini Flash through AI Studio. We provision these so you can start using agents immediately with zero cost. No credit card, no expiration trickery.

What providers work with BYOK?

OpenAI (GPT-4o, o3-mini), Anthropic (Claude Sonnet/Opus), Groq (Llama 3.3 70B), OpenRouter (any model), and Ollama (local models). You control your API key and usage.

Can I switch plans later?

Yes. Start on Free, connect your own key when you're ready, or upgrade to Pro for managed compute. No migration — your workspace and agent state persist.

What does "managed LLM compute" mean in Pro?

We handle the API keys, rate limits, and model selection so you don't have to. You just use the agents — we make sure the AI is always available behind the scenes.
